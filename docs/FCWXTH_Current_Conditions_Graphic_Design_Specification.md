# FCWXTH Current Conditions Graphic Design Specification

## Purpose

Create a clean, broadcast-quality current conditions graphic optimized
for Facebook, livestreams, and mobile viewing.

------------------------------------------------------------------------

# Canvas

-   Aspect Ratio: 16:9
-   Resolution: 1920 × 1080 pixels
-   Background Style: Flat dark charcoal/black with a very subtle
    top-to-bottom gradient.
-   Background Color:
    -   Primary: `#111417`
    -   Alternate: `#101214`

------------------------------------------------------------------------

# Brand Colors

## Gold (Primary Branding)

Use for: - Main title - Current temperature - Degree symbol - Small
separator bullet

Suggested Color:

`#F4B321`

------------------------------------------------------------------------

## Blue (Accent)

Use only for: - CURRENT CONDITIONS - Feels Like

Suggested Color:

`#2F8CFF`

------------------------------------------------------------------------

## White

Use for: - Weather values - Wind speed - Footer text - Station name in
footer

Suggested Color:

`#FFFFFF`

------------------------------------------------------------------------

## Light Gray

Use for: - Card headings - Secondary information - City name

Suggested Color:

`#C8CDD3`

------------------------------------------------------------------------

## Divider Lines

Color:

`#34475A`

Opacity: - 70--80%

Thickness: - 2 px

# Mobile Optimization

Prioritize:

-   Temperature
-   Brand
-   Wind
-   Feels Like

Maintain strong contrast.

Avoid unnecessary decoration.

------------------------------------------------------------------------

# Design Goals

-   Professional broadcast appearance
-   High readability
-   Consistent branding
-   Fast recognition while scrolling
-   Excellent Facebook performance
-   Clean livestream presentation
-   Minimal visual clutter
-   Strong color hierarchy
-   Modern weather dashboard aesthetic
